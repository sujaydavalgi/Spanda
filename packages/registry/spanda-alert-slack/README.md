# spanda-alert-slack

Slack incoming-webhook integration for Spanda Control Center alerting.

## Status

**Experimental** — uses core `spanda-ops` Slack payload formatting when `SPANDA_ALERT_WEBHOOK_URL` points at `hooks.slack.com`.

## Configuration

```bash
export SPANDA_ALERT_WEBHOOK_URL="https://hooks.slack.com/services/..."
```

Install this registry package to document the channel contract in mission manifests and package trust graphs.

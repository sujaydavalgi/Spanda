# Package ecosystem

[← Overview](./README.md)

> **Canonical guides:** [packages.md](../packages.md) · [registry.md](../registry.md) · [official-packages.md](../official-packages.md)

Spanda extends through a **lean core** plus **37 official packages** under `packages/registry/`.

```bash
spanda init my_robot
spanda add spanda-ros2 --version 0.1.0
spanda install && spanda build && spanda test
```

Category tables, registry signing, and publish workflow are documented in the guides above — not repeated here.

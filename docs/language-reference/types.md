# Types

Spanda includes robotics-native types (`Pose`, `Velocity`, `Trajectory`, `Scan`) and standard scalars (`Int`, `Float`, `Bool`, `String`, `Number` with units).

See [spanda-type-system.md](../spanda-type-system.md) and the generated [spanda-reference.md](../spanda-reference.md#type-methods) for the full type method matrix.
